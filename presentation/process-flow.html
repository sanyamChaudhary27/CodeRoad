🔄 CodeRoad Process Flow

1️⃣ User Registration & Authentication
👤 User
Enters username, email, password on registration page
→
☁️ CloudFront CDN
Serves React frontend from S3 bucket globally
→
🔐 FastAPI Backend (EC2)
Validates data, hashes password with bcrypt
↓
💾 PostgreSQL Database
Stores user record with hashed password, initial ELO 300
→
🎫 JWT Token Generation
Creates signed JWT token with user ID, username, email
→
✅ User Authenticated
Receives JWT token, redirected to dashboard
2️⃣ Matchmaking Process
🎮 Player Clicks "Find Match"
Selects arena (DSA/Debug), difficulty preference
→
📋 Join Queue (EC2)
Player added to matchmaking queue with current ELO
→
🔍 Search for Opponent
Finds player within ±200 ELO range in same arena
↓
🤝 Match Created
Both players removed from queue, match record created
→
🤖 AI Challenge Generation
Groq API generates unique problem based on average ELO
→
⚔️ Match Starts
Both players see challenge, timer starts (30-120s)
3️⃣ AI Challenge Generation (Detailed)
📊 Analyze Player Stats
Backend fetches both players' ELO, recent matches, domains
→
🎯 Determine Difficulty
Maps average ELO to difficulty: <500=beginner, 500-1000=intermediate, >1000=advanced
↓
🌐 Groq API Call
Sends prompt with difficulty, domain, recent problems to avoid
→
🧠 Llama 3.1 70B
Generates problem: title, description, test cases, boilerplate code
↓
✅ Validate Response
Parses JSON, validates structure, checks test cases
→
💾 Store Challenge
Saves to database with unique ID, links to match
→
📝 Display to Players
Both players see problem simultaneously
4️⃣ Code Submission & Execution
💻 Player Writes Code
Types solution in browser code editor
→
🚀 Clicks Submit
Code sent to backend via HTTPS POST request
→
📝 Create Submission
Backend creates submission record with code, timestamp
↓
🔒 Sandbox Execution
Code runs in isolated Python subprocess with 5s timeout
→
🧪 Run Test Cases
Executes code against all test cases, captures output
→
✅ Compare Results
Matches actual output with expected output
↓
🛡️ Cheat Detection
XGBoost model analyzes code style, timing, complexity
→
💾 Update Submission
Stores results: passed/failed, execution time, cheat score
→
📊 Show Results
Player sees test results, can submit again
5️⃣ Match Conclusion & ELO Update
⏱️ Timer Expires OR Both Done
Match ends when time runs out or both players signal done
→
🏆 Determine Winner
Compare test cases passed, execution time, submissions count
↓
📈 Calculate ELO Changes
Winner gains ELO, loser loses ELO based on rating difference
→
💾 Update Database
Updates player ratings, match stats, win/loss records
→
🎉 Show Results
Players see winner, ELO changes, match summary
6️⃣ AWS Infrastructure Flow
🌐 User Browser
Requests coderoad.online
→
🌍 Route 53 DNS
Resolves domain to CloudFront distribution
→
⚡ CloudFront CDN
Serves cached frontend from nearest edge location
↓
📦 S3 Bucket
Stores React build files (HTML, CSS, JS)
→
🖥️ Frontend Loaded
React app renders in browser
↓
📡 API Requests
Frontend makes HTTPS calls to backend
→
🔒 Security Group
Firewall allows ports 80, 443, 8000
→
🖥️ EC2 Instance
t3.small running Ubuntu + FastAPI
↓
🔄 Nginx Reverse Proxy
Routes requests to backend, handles SSL
→
🐍 FastAPI Backend
Processes request, queries database
→
🐘 PostgreSQL (Docker)
Returns data to backend
↓
💾 EBS Volume
Persistent storage for database, logs, code
→
✅ Response to User
JSON data displayed in React UI
