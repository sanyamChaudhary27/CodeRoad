# CodeRoad - Hackathon Presentation Materials

This folder contains all presentation materials for the AI for Bharat hackathon.

## 📁 Files Included

### 1. `idea.md`
Comprehensive project documentation including:
- Brief about the idea (~100 words)
- Why AI is highly required
- How AWS services are used
- Value AI adds to user experience
- Technologies utilized
- Estimated implementation cost
- Prototype performance benchmarking

### 2. `features.html`
Interactive HTML page showcasing all features:
- Core features (1v1 battles, AI generation, ELO system)
- Arena types (DSA, Debug, DBMS, UI)
- AI-powered features
- User experience features
- Technical features
- **Usage**: Open in browser and take screenshots

### 3. `process-flow.html`
Complete process flow diagrams showing:
- User registration & authentication flow
- Matchmaking process
- AI challenge generation (detailed)
- Code submission & execution
- Match conclusion & ELO update
- AWS infrastructure flow
- **Usage**: Open in browser and take screenshots

### 4. `architecture.html`
System architecture diagram with layers:
- User layer
- AWS CDN layer (CloudFront, S3)
- Frontend layer (React)
- AWS infrastructure layer (EC2, VPC, Security Groups)
- Backend layer (FastAPI, services)
- AI/ML layer (Groq, XGBoost)
- Data layer (PostgreSQL)
- **Usage**: Open in browser and take screenshots

### 5. `wireframes.html`
UI wireframes for all major pages:
- Landing page
- Dashboard
- Arena (active match)
- Leaderboard
- User profile
- **Usage**: Open in browser and take screenshots

## 🎯 How to Use

### For Screenshots:
1. Open each HTML file in a modern browser (Chrome/Firefox recommended)
2. Press F11 for fullscreen mode
3. Take screenshots using:
   - Windows: Win + Shift + S
   - Mac: Cmd + Shift + 4
4. Save screenshots for presentation slides

### For Documentation:
- Use `idea.md` content directly in presentation slides
- Copy-paste relevant sections as needed
- All metrics and numbers are accurate as of March 6, 2026
